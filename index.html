Get started in three steps

1. Install

bash <(curl -sSL https://browserbox.io/install.sh) install

Windows: irm https://browserbox.io/install.ps1 | iex

2. Get a key

Choose commercial or non-commercial licensing and receive your key by email.

3. Activate

bbx certify YOUR_KEY

bbx setup && bbx run
